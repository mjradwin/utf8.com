Character Sets

The character set for UTF-8 is UTF-8. So, for example, your HTTP header might look like:

Content-Type: text/html;charset=UTF-8

Or, in HTML parlance:

<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">

See the IANA's Character Sets document for the official names for character sets that may be used in the Internet and may be referred to in Internet documentation.
